# Skincare Recommendation Web App

A full-stack web application that helps users find the right skincare products based on their specific needs.

## Features
- Collects user data on skin type, skin concerns, budget and desired results
- Generates personalized skincare product recommendations
- Provides Amazon purchase links for recommended products
- Clean and user-friendly interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Design: Figma

## How to Run
1. Install XAMPP
2. Import the database file in MySQL Workbench
3. Open index.html in your browser
